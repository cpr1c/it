Справочник предназначен для примера использования ряда подсистем.

В подсистеме Обмен данными справочник демонстрирует пример объекта, который синхронизируется с другими информационными базами.

В подсистеме Поиск и удаление дублей справочник демонстрирует возможность замены элементов для объектов, связанных по владельцу.

В подсистеме Управление доступом справочник показывает пример типичного использования логики ограничения.

  • Список ограничивается по двум видам доступа:
    • Организации (проверяется значение реквизита Организация);
    • Демо: Группы партнеров (проверяется значение реквизита Партнер);
    • все значения должны быть разрешенными одновременно (в одной из групп доступа).
